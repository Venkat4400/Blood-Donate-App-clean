
# 🔴 Blood Donation Web App

## 📌 About the Project

Blood Donation Web App is a full-stack web application that connects blood donors with recipients. Users can register, log in, search donors by blood group and location, and manage donation requests securely.

The application is built using modern frontend technologies and a backend powered by Supabase for authentication and database management.

---

## 🚀 Features

* 🔐 User Authentication (Sign Up / Login)
* 🩸 Search Donors by Blood Group
* 📍 Location-based Filtering
* 🗂 Secure Database with Row Level Security (RLS)
* 📱 Fully Responsive UI
* ⚡ Fast Build using Vite
* 🎨 Styled with Tailwind CSS
* 🔄 Automatic Deployment using GitHub Actions
* 🌍 Hosted on GitHub Pages

---

## 🔧 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Venkat4400/Blood-Donate-App-clean.git
```

2. Navigate to project folder:

```bash
cd Blood-Donate-App-clean
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
